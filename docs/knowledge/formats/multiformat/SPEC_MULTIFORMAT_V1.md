# SPEC_MULTIFORMAT_V1.md

SPEC_ID: MULTIFORMAT_V1
USE_WHEN: мультиформат, push, sms, и тд.
FORMAT_TAGS: мультиформат, пуш, смс, лендинг, яндекс_карты, онбординг, табы, стики-баннер, стики-баннер с таймером, бегущая_строка

## Goal
Сформировать согласованный набор текстов для разных каналов с единым оффером, но гибкой тональностью под каждый канал:
- Push / Push Reminder / Last Call
- SMS / SMS Reminder / Last Call  
- Лендинг, бегущая строка, стики, табы
- Яндекс карты, онбординг

## Hard constraints

### 1. Лимиты длины (Строгие)
Если текст превышает лимит — это критическая ошибка (ERROR).

**Все форматы(пуш, смс, и тд):**
* Запрещено сокращать слова.
* Для прогрессивной механики, если не удается вписать её целиком, оставить только максимальную скидку.

**ПУШ (Анонс / Напоминание / Последний звонок):**
* Заголовок: ≤ 32 символов (с пробелами).
* Подзаголовок: ≤ 60 символов (с пробелами).

**СМС:**
* Текст: ≤ 78 символов (+ фраза «по карте №12345»).
* *Важно:* Фразу «по карте №12345» важно учитывать в смысловой нагрузке.

**ЯНДЕКС КАРТЫ:**
* Текст: 150–300 символов.

**ОНБОРДИНГ:**
* Заголовок: ≤ 40 символов.
* Подзаголовок: ≤ 60 символов.
* Кнопка: короткая, активная (1-2 слова).

**ТЕКСТ НА LP (Landing Page):**
* Заголовок: ≤ 47 символов.
* Описание: ≤ 428 символов (рекомендация: держись в пределах 375).

**ТАБЫ (Фильтры товаров):**
* Текст: 250–500 символов (оптимально 350–400).

**СТИКИ-БАННЕР БЕЗ ТАЙМЕРА:**
* Текст: ≤50 символов (1 строка).

**СТИКИ-БАННЕР С ТАЙМЕРОМ:**
* Текст: ≤58 символов (включая тег `[таймер]`).
* Правило: фраза должна логично требовать продолжения (не завершенная мысль, а подводка к таймеру).

**БЕГУЩАЯ СТРОКА:**
* Текст: ≤40 символов.


### 2. Правила упоминания алкоголя
**СМС:** Упоминание алкоголя запрещено.
* Некорректно: «Купи вино», «Алкоголь дешево», «Игристое», «Крепкое», «Красное».
* Корректно: «коллекция», «ассортимент», «хиты», «подборка», «выгода», «подарки».

**Все форматы:** Можно упоминать категории (вино, игристое, крепкое), но в культурном контексте без агрессивных продаж. 

### 3. Правила контента и стиля

**Единый месседж:** условия, цифры скидок и сроки должны быть идентичны во всех форматах.

**Гибкая тональность по каналам:**

**Push/SMS:**
* Стиль: прямой, краткий, срочный.
* CTA: действенные фразы — «Заказать прямо сейчас», «Поторопитесь», «Не упустите».
* Тон: акцент на выгоду и ограниченность времени.
* Длина: максимальная краткость, без лишних деталей.

**Лендинг/Бегущая строка/Стикеры:**
* Стиль: метафоричный, описательный, вовлекающий.
* CTA: информативные, встроенные в контекст — «Найдите идеальное вино», «Узнайте больше о Тоскане», «Откройте для себя».
* Тон: эмоциональный, образный, акцент на преимуществах и настроении.
* Длина: можно использовать больше слов для создания атмосферы.

**Яндекс карты/Онбординг:**
* Стиль: прямой, но дружелюбный.
* CTA: емкие, понятные — «Выбрать», «Заказать», «Перейти к подборке».
* Тон: приглашение, знакомство, акцент на удобство и простоту действия.

**Юридика:**
* Не называть вино «напитком».
* Не обещать пользу здоровью.
* CTA: «Заказать», «Выбрать», «Открыть» (не «Купить»).

## PREFLIGHT_CHECKS
1. ✓ Соблюдена длина по каждому формату (лимиты из раздела 1).
2. ✓ Алкоголь упоминается корректно:
   - Есть контекст (вина Тосканы, коллекция, для особых случаев).
   - Нет агрессивных призывов типа «Купи алкоголь».
3. ✓ CTA соответствует каналу:
   - Push/SMS: прямые и срочные («Заказать сейчас», «Поторопитесь»).
   - Лендинг/Стикеры: вовлекающие («Найдите идеальное», «Узнайте больше»).
   - Яндекс/Онбординг: емкие и понятные («Выбрать», «Заказать»).
4. ✓ Месседж согласован: цифры скидок и сроки идентичны во всех форматах.
5. ✓ Тональность релевантна каналу:
   - Push/SMS: краткость, срочность, акцент на выгоду.
   - Лендинг/Стикеры: метафоричность, эмоциональность, образность.
   - Яндекс/Онбординг: дружелюбие, приглашение.
6. ✓ Стики с таймером: текст заканчивается на `[таймер]`.
7. ✓ Пуш Последний звонок: есть ощущение срочности (метафора финала).
8. ✓ Нет сокращенных слов.
9. ✓ Текст повествовательный, не рваный (с глаголами). Соблюдаются правила современного русского языка. 
10.✓ Cоблюдена тональность из SW_GLOBAL_RULES_V2.
11. Избегай физически некорректных метафор.


## Примеры (Patterns)

**Пример 1: Push Анонс (прямой стиль)**
Заголовок: Скидки до -40% стартовали
Подзаголовок: Заказывайте хиты прямо сейчас — акция ограничена

**Пример 2: SMS Анонс (краткий и срочный)**
Текст: До -40% на хиты и новинки — только до 15 августа (+ фраза «по карте №12345»)

**Пример 3: Лендинг (метафоричный стиль)**
Заголовок: Бокалы, полные лета
Описание: Откройте для себя солнечные вина Тосканы и освежающие игристые из Франции. Скидки до -35% на сезонную подборку — идеальное дополнение к вашим летним вечерам.

**Пример 4: Бегущая строка (ритм и энергия)**
Текст: Эта строка бежит на пикник — до -50% почти на всё

**Пример 5: Стики-баннер с таймером**
Текст: Скидки до -50% обнулятся через [таймер]
Текст: До -50% на топ-позиции — активны ещё [таймер]

**Пример 6: Пуш Последний звонок (срочность)**
Заголовок: Пошли титры
Подзаголовок: Последний шанс забрать хиты со скидкой до -50%

**Пример 7: Онбординг (приглашение)**
Заголовок: Вау-скидки
Подзаголовок: До -50% на подарки к 8 Марта
Кнопка: Выбрать

**Пример 8: Яндекс карты (дружелюбный и информативный)**
Текст: Вина для особенных случаев теперь доступнее. Скидки до -35% на избранные позиции из Италии и Франции. Заказывайте прямо из приложения. Предложение действует до 20 августа.
